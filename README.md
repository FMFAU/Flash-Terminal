# Flash-Terminal

**A universal terminal solution designed to replace traditional terminals like MySys32 and more.**

---

## Overview

Flash-Terminal aims to be your all-in-one terminal, combining essential tools and features to streamline your workflow without needing multiple terminal applications. It integrates package installers, macros, system commands, and much more in a single, unified interface.

---

## Features

* Universal terminal interface
* Built-in package installers
* Customizable macros
* Core system commands support
  
---

## Installation

* Run in the installer, and thats it!

---
